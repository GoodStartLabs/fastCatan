# fastCatan — pinned Python dependencies (cross-platform: macOS + Linux).
#
# Versions captured from the known-good Linux/anaconda env (CPython 3.12,
# numpy 1.26). Install this SAME set on every machine so trained checkpoints and
# the native build stay compatible across macOS and Linux.
#
# Setup (any platform):
#   python3 -m venv .venv && source .venv/bin/activate   # use CPython 3.12
#   pip install -U pip
#   pip install -r requirements.txt
#   pip install -e . --no-build-isolation --config-settings=editable.rebuild=true
#
# Use CPython 3.12: the native extension builds as cp312, and SB3 checkpoints are
# interpreter-version-sensitive.
#
# !!! numpy is pinned to 1.26.x ON PURPOSE. Checkpoints pickled under numpy 2.x do
# NOT load under numpy 1.x (the numpy._core rename + BitGenerator / gym-Space
# pickle breakage) and vice-versa. If you ever bump numpy, do it on BOTH machines
# at once and re-save every checkpoint. (This is exactly what bit the shaped
# checkpoints trained on the Mac under numpy 2.x.)
